TIXO is an interpreter for XO of projects created with the TICO Editor, part of the TICO Porject (http://www.proyectotico.es). TIXO is an abbreviation for TICO for XO, and TICO is an abbreviation for Talberos Interactivos para la Comunicacion.

TICO was developed to work in a Sugar enviroment, but is capable to work under any linux machine with python installed. To run outside Sugar you just need to lunch TicoInterpreter.py
